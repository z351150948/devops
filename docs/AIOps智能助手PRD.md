# AIOps 智能助手 PRD

## 1. 产品定位

- 产品名称：AIOps 智能助手
- 产品目标：为 SxDevOps 平台提供一个右下角可随时唤起的聊天机器人，用自然语言完成资源查询、告警解释、故障关联分析、任务自动生成与触发。
- 管理员目标：提供机器人配置中心，管理模型 API、Key、MCP、Skill、提示词与执行安全策略。

## 2. 目标用户

- 普通用户：查询自己有权限看到的资源、告警、日志与任务信息
- 运维管理员：辅助排障、生成并执行主机任务
- 平台管理员：配置模型、MCP、Skill 与安全策略

## 3. 核心场景

- 资源查询：查询主机、CMDB、多云、K8s、Docker、中间件、IaC 资源
- 告警咨询：查询当前未确认告警、严重告警、按主机或服务筛选告警
- 关联分析：结合告警、事件墙、日志、链路、变更与任务记录做排障分析
- 自动化任务：生成主机巡检、服务检查、批量命令、Playbook 执行任务
- 管理配置：配置模型接入、MCP、Skill、Prompt 与执行安全

## 4. 功能范围

### In Scope

- 全局右下角悬浮聊天机器人
- 聊天会话与历史消息
- 平台资源与告警问答
- 跨模块关联分析
- 基于现有任务中心的任务草稿生成与确认执行
- 管理员配置中心
- MCP 与 Skill 接入配置框架
- 审计与权限控制

### Out Of Scope

- 第一阶段不做可视化工作流编排器
- 第一阶段不做多智能体自治编排
- 第一阶段不默认开启自动闭环处置
- 第一阶段不接入外部 IM 机器人

## 5. 用户端功能

### 5.1 右下角聊天机器人

- 任意主页面右下角显示悬浮按钮
- 点击后弹出聊天面板，支持最小化、关闭、展开
- 支持新建会话、历史会话、快捷提问、复制回答
- 回答结构尽量包含：结论、证据、建议动作、跳转入口

### 5.2 资源信息咨询

- 查询范围：主机、CMDB、多云、K8s、Docker、中间件、IaC
- 支持按系统、环境、负责人、资源类型、状态、风险等级筛选
- 返回摘要数字、列表卡片和关联跳转

### 5.3 告警信息咨询

- 查询当前告警、未确认告警、严重告警
- 支持按主机、服务、系统、时间窗口筛选
- 返回告警摘要、影响对象、推荐排查入口

### 5.4 关联分析排障

- 聚合告警、事件墙、链路、日志、最近任务与部署变更
- 输出最可能原因、证据链、排查步骤与风险提示
- 第一阶段以规则 + 工具查询 + LLM 总结为主

### 5.5 自动编写和触发任务

- 复用现有 Host Task / Host Task Template / Host Task Schedule 能力
- 自动生成任务标题、描述、任务类型、命令、目标主机、执行方式、超时和策略
- 所有执行动作默认先预览再确认
- 高风险命令必须二次确认

## 6. 管理员配置中心

- 页面名称：AIOps 配置
- 可配置内容：
  - 模型 API、Key、Base URL、默认模型、备用模型、温度、超时
  - System Prompt、欢迎语、建议问题
  - MCP 服务配置
  - Skill 配置与启停
  - 执行安全策略
  - 审计开关与保留策略

## 7. 权限设计

- `aiops.chat.view`：查看机器人入口并发起对话
- `aiops.chat.analyze`：允许做关联分析
- `aiops.task.generate`：允许生成任务草稿
- `aiops.task.execute`：允许通过机器人触发任务
- `aiops.config.view`：查看 AIOps 配置页
- `aiops.config.manage`：管理模型、MCP、Skill、Prompt 与安全策略
- `aiops.audit.view`：查看机器人会话与工具调用审计

权限原则：

- 机器人只能调用用户已有权限覆盖的数据与动作
- 机器人不能成为权限放大器
- 问答权限与执行权限分离

## 8. 技术方案

- 前端：在全局布局挂载 `AIOpsChatWidget`
- 后端：新增 `backend/aiops/` app
- 通信：MVP 先使用常规 HTTP，会话消息支持增量扩展到 SSE / WebSocket
- 工具层：资源查询、告警查询、事件查询、日志查询、链路查询、任务生成、任务执行
- 模型层：优先支持 OpenAI-Compatible 接口，同时保留本地规则回退

## 9. MVP 范围

- 右下角聊天机器人浮窗
- 资源 / 告警 / 事件 / 任务信息问答
- 告警关联分析摘要
- 任务中心任务草稿生成
- 任务确认执行
- 管理员模型配置页
- OpenAI-Compatible 模型接入
- 基础会话审计
- RBAC 权限接入

## 10. 验收标准

- 用户能在任意主页面右下角打开机器人
- 用户能查询自己权限范围内的资源和告警
- 机器人回答附带平台内证据来源
- 用户能生成主机任务草稿
- 具备执行权限的用户能确认并触发任务
- 管理员能配置模型 API、Key、Base URL、默认模型
- 管理员能配置 MCP 与 Skill 开关
- 所有执行动作可审计、可追溯
